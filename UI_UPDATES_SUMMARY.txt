🔧 TECHNICAL CHANGES

1. Layout Structure
   - Changed from equal-width panels to flexible right panel
   - Left panel: Fixed 520px width (compact video + snapshot)
   - Right panel: Flexible width (grows to fill space)
   - Uses QWidget containers instead of direct layouts

2. Styling Improvements
   - Consistent color scheme across all sections
   - Professional borders and shadows (where supported)
   - Better typography with varied font sizes
   - Improved visual hierarchy with section headers

3. Input Fields
   - Unified 32px height for consistency
   - Focus state changes border to section color
   - Light gray background (#f9fafb) for clean look
   - 2px borders that highlight on focus

4. Buttons
   - Professional styling with hover/press states
   - Consistent font (Arial, 9pt, Bold)
   - Varied heights for visual priority (Capture: 45px, others: 40-42px)
   - Smooth color transitions on interaction

5. Removed CSS Properties
   - Removed: box-shadow (not supported by PyQt5 stylesheets)
   - Kept: All other styling properties
