🔍 KEY CONCEPTS:

Card ID (Parking Ticket):
  • Operator sees on parking ticket
  • Manually typed by operator
  • Example: CARD-2024-001234, PK-2024-5678, ABC-12345
  • Used to identify parking transaction
  • Human-readable ticket number

License Plate (Vehicle ID):
  • Automatically extracted from camera image
  • Recognized by OCR algorithm
  • Example: ABC-123-XYZ, AB12CDE, 12AB-XYZ
  • Identifies the vehicle
  • May appear multiple times (same vehicle, different times)

Why Both?
  • Card ID = Administrative/billing identifier
  • License Plate = Vehicle identifier
  • Allows proper fee calculation
  • Enables vehicle history tracking
  • Prevents data errors
