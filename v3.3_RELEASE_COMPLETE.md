## Technical Specifications

### Hardware Support
- Arduino Uno/Mega with RFID reader
- ESP32 with RFID/NFC module
- Any device capable of serial JSON output
- Standard USB-to-Serial adapter compatible

### Software Requirements
- Python 3.7+
- PyQt5 5.15.9+
- OpenCV 4.8.0+
- YOLO 8.0.238+
- EasyOCR 1.6.2+
- pyserial 3.5+

### Performance Metrics
- Serial communication: ~10-20ms latency
- YOLO detection: ~200-500ms per frame
- OCR extraction: ~500-1000ms per crop
- Total auto check-in/out: ~1.5-2 seconds
- CSV write: ~5-10ms per record
- UI refresh: ~20-30ms per update

### Data Format

**Incoming JSON from Arduino:**
```json
{
  "event": "CARD_SCAN",
  "uid": "UNIQUE_CARD_ID",
  "gate": "ENTRY or EXIT",
  "slot": parking_slot_number
}
```

**Outgoing CSV Record:**
```csv
Card ID,License Plate,Time In,Time Out,Slot Number,Status,Fee (VND)
A1B2C3D4,ABC123XYZ,2026-01-20 10:30:45,2026-01-20 11:45:30,1,OUT,25000
```

**Outgoing Excel File:**
- Professional formatting with colored headers
- Color-coded status (green IN, red OUT)
- Right-aligned fee numbers
- Frozen header row
- Column auto-sizing

## Installation & Deployment

### Step 1: Install Dependencies
```bash
pip install -r requirements.txt
```

### Step 2: Connect Hardware
1. Plug Arduino/ESP32 into USB port
2. Verify connection with: `python -m serial.tools.list_ports`

### Step 3: Run Application
```bash
python main_parking.py
```

### Step 4: Configure Serial Connection
1. Select port from dropdown (should auto-detect)
2. Select baud rate (usually 9600)
3. Click "🔗 CONNECT"
4. Wait for "🟢 Connected" indicator

### Step 5: Verify System
1. Scanner should accept RFID cards
2. Check console for "Card scanned: ..." messages
3. Verify UI updates automatically
4. Check `/captured_images/` folder for frame captures

## Usage Workflow

### Automatic (with RFID Scanner)
```
1. Vehicle arrives at entrance
2. Driver scans RFID card
3. System captures license plate automatically
4. Entry recorded in database (1-2 seconds)
5. Vehicle enters

6. Vehicle exits parking
7. Driver scans same RFID card
8. System captures plate, finds entry, calculates fee
9. Exit recorded with fee (1-2 seconds)
10. Vehicle leaves
```

### Manual Fallback (if scanner down)
```
1. Open application
2. Point camera at license plate
3. Click "📸 CAPTURE LICENSE PLATE"
4. Enter Card ID and Slot manually
5. Click "✅ CHECK IN"
6. To exit: Enter plate, click "🏁 CHECK OUT"
```

## Database Structure

### In-Memory Records
```python
parking_records = [
    {
        'card_id': 'A1B2C3D4',           # RFID card ID
        'license_plate': 'ABC123XYZ',    # Auto-extracted from OCR
        'time_in': datetime(...),        # Entry timestamp
        'time_out': datetime(...),       # Exit timestamp (null if IN)
        'slot': '1',                     # Parking slot number
        'status': 'IN' or 'OUT',         # Current status
        'fee': 0,                        # Calculated fee (0 if IN)
    },
    ...
]
```
