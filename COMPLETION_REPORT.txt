🔄 SYSTEM ARCHITECTURE
═══════════════════════════════════════════════════════════════════════════════

Threading Model
  Main Thread        → PyQt5 UI Event Loop (responsive GUI)
  Worker Thread      → Camera capture + YOLO + OCR (non-blocking)
  Signal Flow        → Frame ready + License detected signals

Processing Pipeline
  1. Capture frame (1280x720, 30 FPS)
  2. Run YOLO detection
  3. Extract bounding boxes
  4. Crop license plate regions
  5. Convert to grayscale & threshold
  6. Run EasyOCR recognition
  7. Apply character correction
  8. Validate format (AA9999A)
  9. Update UI & save records

Data Flow
  Detection → Signal → Main Thread → Update Table → Save CSV → Display Status
