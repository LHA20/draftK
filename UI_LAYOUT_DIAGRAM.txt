USER WORKFLOW:
═════════════════════════════════════════════════════════════════════════════

CHECK-IN WORKFLOW:
┌────────────────────────────────────────────────────────────────────────┐
│ 1. User watches live video feed (top-left, 380×260px)                  │
│ 2. User clicks "📸 CAPTURE LICENSE PLATE" button (cyan)                │
│    ├─ System captures current video frame                              │
│    ├─ Runs YOLO license plate detection                                │
│    ├─ Extracts text using EasyOCR                                      │
│    ├─ Displays snapshot in area below video                            │
│    └─ Shows extracted text in purple box                               │
│ 3. User enters Card ID in input field                                  │
│ 4. User enters Parking Slot Number in input field                      │
│ 5. User clicks "✅ CHECK IN (IN)" button (green)                       │
│    ├─ System validates inputs                                          │
│    ├─ Creates record with status = "IN"                                │
│    ├─ Saves to parking_records.csv                                     │
│    ├─ Updates records table                                            │
│    └─ Shows "Vehicle checked in" message                               │
│ 6. System ready for next vehicle                                       │
└────────────────────────────────────────────────────────────────────────┘

CHECK-OUT WORKFLOW:
┌────────────────────────────────────────────────────────────────────────┐
│ 1. User enters License Plate number in CHECK OUT section (or auto-fill) │
│ 2. User clicks "🏁 CHECK OUT (OUT)" button (red)                       │
│    ├─ System looks up license plate in CSV                             │
│    ├─ Validates vehicle exists and status = "IN"                       │
│    ├─ Calculates parking duration                                      │
│    ├─ Calculates fee:                                                  │
│    │  • If duration ≤ 2 hours: 20,000 VND                              │
│    │  • If duration > 2 hours: 20,000 + (hours - 2) × 10,000 VND       │
│    ├─ Updates record: status = "OUT", Time Out = now, Fee = amount     │
│    ├─ Saves to CSV                                                     │
│    ├─ Updates records table                                            │
│    ├─ Displays fee in yellow box                                       │
│    └─ Shows confirmation message                                       │
│ 3. System ready for next vehicle                                       │
└────────────────────────────────────────────────────────────────────────┘

DATA MANAGEMENT:
┌────────────────────────────────────────────────────────────────────────┐
│ 1. View all parking records in scrollable table (7 columns)            │
│ 2. Click "💾 DOWNLOAD EXCEL REPORT" button (purple)                    │
│    ├─ File dialog opens                                                │
│    ├─ User selects save location                                       │
│    ├─ System generates Excel file with:                                │
│    │  • Professional formatting                                        │
│    │  • Bold headers                                                   │
│    │  • Proper column widths                                           │
│    │  • Currency formatting for fees                                   │
│    │  • All parking records                                            │
│    └─ File saved to user's location                                    │
│ 3. System maintains CSV backup (auto-saved)                            │
└────────────────────────────────────────────────────────────────────────┘
